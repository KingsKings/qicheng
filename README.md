# 启程 - ADHD 专注力辅助应用

一款面向 ADHD 群体的轻量级任务管理与专注力训练工具，帮助用户降低任务启动阻力，建立稳定的节奏感。

## 功能特点

- **任务管理** - 简单易用的待办事项管理，支持快速添加、分解任务
- **专注计时** - 内置冥想与专注计时器，辅助集中注意力
- **成就系统** - 记录每日完成情况，可视化进度追踪
- **ADHD 自测** - 基于专业量表的自我评估工具

## 技术栈

- Vue 3 + Vite
- 原生 CSS（支持深色模式）

## 开发运行

```bash
npm install
npm run dev
```
